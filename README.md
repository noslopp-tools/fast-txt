# Fast Txt Releases

This repository contains release binaries for Fast Txt.

## Downloads

Navigate to [Releases](https://github.com/noslopp-tools/fast-txt/releases) to download the latest version for your platform.

### Windows
- Download the `.exe` installer from the latest release

### Android
- Download the `.apk` from the latest release
